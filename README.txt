	How to use this Script?

_______________________________________________________________________________

1.) Upload the content to your Server !
2.) give the Main folder, Databases and includes CHMOD 0777
3.) open the checkmodul.php and look if you have the requirements for this script
4.) use the index.php and push "Create NOW" to get a Hosting folder.
5.) now push the SETUP GO button if you have all data together.
6.) use "all files" of the "new folder" on your server as "main files" and replaced this.
7.) have fun!
